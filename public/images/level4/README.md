# Level 4: City Metropolis — Parallax Assets

**Milestone:** 8.B.1
**Status:** CEO approved 2026-03-10

City Metropolis theme assets for Level 4. Kaladesh-inspired metropolitan architecture viewed from strict top-down overhead perspective.

## Assets

| File | Scroll Speed | Description |
|------|--------------|-------------|
| `parallax_far.png` | 0.3x | Atmospheric haze—copper/brass building silhouettes through blue-gray mist |

## Style Reference

- **Source:** `docs/concepts/p8_mocks/8_a3_themes/theme_city_metropolis_sample.png`
- **Palette:** Warm metallics (copper #B87333, gold #B5A642, brass), aether-blue #00FFFF

- **Tileable:** All layers tile vertically for infinite scroll
