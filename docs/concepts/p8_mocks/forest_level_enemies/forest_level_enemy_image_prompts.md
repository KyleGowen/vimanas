# Forest Level Enemy Image Prompts

**Phase 8 · Visual Design Agent**

Self-contained prompts for generating the five forest-level enemy sprites. Use with an AI image generator. Each prompt specifies strict top-down view, palette, silhouette, and style. Output filenames: scout_forest.png, medium_forest.png, elite_forest.png, miniboss_forest.png, boss_forest.png.

## scout_forest.png

Game sprite, strict top-down bird's-eye view only—enemy seen from directly above, silhouette readable from above. Small insectoid alien swarm unit: elongated arrowhead body with a mantis-like prow; twin-mandible cluster at the nose (two angular prongs forming a V pointing forward); swept, slightly splayed carapace wings; segmented, angular exoskeleton. No rounded cockpit—creature read, not craft. Palette: amber #FFBF00 (carapace primary), olive #6b8e23 (segments, accents), dark brown #3d2914 (underside, depth, joints). Sophisticated 2D illustrated style: clean shading, metallic carapace texture, crisp rendering—NOT pixel art, NOT 16-bit. Kaladesh-adjacent forest-organic. Transparent background or solid dark neutral #1a1a2e. Sprite size target: 192–256 px. Single enemy centered in frame.
