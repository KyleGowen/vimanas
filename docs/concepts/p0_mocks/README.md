# P0 Investor Pitch Mocks

Phase 0 deliverables for investor pitch. Each subfolder contains a P0 mock and its related assets.

| Folder | Contents |
|--------|----------|
| **p0_1_ships** | Ship mocks (Sparrow, Turtle, Wolf, Dragon). Each ship in own subdir: sparrow/, turtle/, wolf/, dragon/. Pilot-style mocks canonical (CEO approved 2025-03-02). |
| **p0_2_pilots** | Pilot mocks (Speed, Weapon, Defensive, Neutral). Portraits, visual briefs. |
| **p0_3_levels** | Level mocks (Forest, Industrial, Sky). Parallax concepts, terrain framing. |
| **p0_4_boss** | Boss fight mocks (Root-Seeker, Conduit-Crawler). Encounter briefs, image prompts. |
| **p0_5_title_screen** | Title screen mock. Concept, image prompts. |
| **p0_6_narrative** | Sample narrative (opening). |

**Gate:** CEO must explicitly approve each mock before use in pitch or production.
