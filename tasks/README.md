# Tasks

## Structure

- **active/** — Task files for in-progress or pending work. Reference when planning or executing milestones.
- **archive/** — Completed task files. Phases 1–6 (through Turtle, Wolf, Dragon) and related features are archived. Kept for history; not actively referenced.

## Archive Convention

When a milestone or phase is marked Done in [plans/roadmap.md](../plans/roadmap.md), move its task file(s) from `active/` to `archive/`. Keep in `active/` only:

- Pending design/tech tasks (e.g. 6.H.1–6.H.3, 6.S.1–6.S.2)
- P0 mocks or deliverables awaiting CEO sign-off (e.g. P0_007)
